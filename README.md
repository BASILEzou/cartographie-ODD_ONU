# cartographie-ODD_ONU
Projet de cartographie des ODD de l'ONU en France métropolitaine de 1954 à 2024, à l'échelle départmenet ou région.
Il faut télécharger tout les fichiers, puis lancer le code de création des dictionnaires, puis lancer le app_main. 
Attention il faut changer les chemins d'accès aux fichiers dans votre code.
